# audio-cutter-subtitles
Cut the voice file corresponding to the subtitles into sections.

## input
- input.wav

## vtt format
- input.vtt
```
WEBVTT

1
00:01.000 --> 00:04.000
Never drink liquid nitrogen.

2
00:05.000 --> 00:09.000
- It will perforate your stomach.
- You could die.

...
```

## output
- output.0001.wav
- output.0002.wav
- ...
